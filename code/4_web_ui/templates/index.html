Cổng Quản trị Hệ điều hành Linux

Hệ thống cung cấp giao diện quản trị tập trung cho ba nhóm chức năng của đồ án môn học “Lập trình nhân Linux”: quản lý hệ thống bằng kịch bản Shell, lập trình C tương tác với hệ điều hành, và xây dựng – quản lý module nhân. Mọi thao tác được thực thi trực tiếp trên máy chủ Ubuntu.

Phần 1 · Lập trình Shell
  • Quản lý tệp tin
  • Lập lịch tác vụ (crontab)
  • Thiết lập thời gian hệ thống
  • Cài đặt / gỡ bỏ gói phần mềm
Phần 2 · Lập trình C
  • Quản lý tiến trình
  • Quản lý tệp tin (C)
  • Thông tin mạng
  • Giao tiếp Socket TCP
Phần 3 · Module nhân
  • Biên dịch module nhân
  • Nạp / gỡ module
  • Xem nhật ký nhân (dmesg)
Hướng dẫn sử dụng

Chọn chức năng tương ứng ở thanh điều hướng phía trên hoặc danh mục bên trái. Một số thao tác (thiết lập thời gian, cài/gỡ gói phần mềm, nạp/gỡ module nhân) yêu cầu quyền quản trị sudo; vui lòng khởi chạy máy chủ với quyền phù hợp.

Chức năng quản lý tệp tin hoạt động trong thư mục an toàn ~/ltnln_workspace nhằm tránh ảnh hưởng tới hệ thống.
